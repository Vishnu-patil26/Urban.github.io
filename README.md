# 🌆 Urban Web Project

## 🚀 Overview

Urban is a front-end web project focused on building a clean and modern user interface using core web technologies. It demonstrates design structure, layout skills, and responsive web development.

## 🎯 Objective

To develop a visually appealing and structured website layout while improving frontend development skills.

## 🧑‍💻 Features

* Responsive layout
* Clean UI design
* Structured sections for content
* Organized asset management (images, styles, scripts)

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
